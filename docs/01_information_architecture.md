# Componi - 정보 구조도 (Information Architecture)

## 1. 직원용 모바일 앱 (Employee App)

```
[홈 대시보드]
├── 오늘의 근무 현황
│   ├── 출근/퇴근 버튼
│   ├── 현재 근무 시간
│   └── 이번 주 누적 근무시간
├── 빠른 메뉴
│   ├── 연차 신청
│   ├── 급여 명세서
│   └── 증명서 발급
└── 공지사항 (최신 3개)

[근태관리] ─────────────────────────────
├── 출퇴근 체크
│   ├── GPS 기반 출근
│   ├── Wi-Fi 기반 출근
│   └── 출퇴근 기록 수정 요청
├── 근무 현황
│   ├── 일별 근무 기록
│   ├── 주간 근무 현황 (52시간 게이지)
│   └── 월간 근무 캘린더
└── 휴가/연차
    ├── 잔여 연차 현황
    ├── 휴가 신청
    ├── 신청 내역 조회
    └── 결재 현황

[급여관리] ─────────────────────────────
├── 이번 달 급여
│   ├── 급여 명세서 상세
│   └── PDF 다운로드
├── 급여 히스토리
│   └── 월별 급여 명세서 목록
└── 통상임금 정보

[전자계약] ─────────────────────────────
├── 대기 중인 계약
│   ├── 근로계약서
│   ├── 비밀유지서약서
│   └── 기타 서약서
├── 서명하기 (Canvas)
└── 완료된 계약
    └── 계약서 PDF 다운로드

[인사서류함] ───────────────────────────
├── 증명서 신청
│   ├── 재직증명서
│   ├── 경력증명서
│   └── 원천징수영수증
├── 발급 내역
└── PDF 다운로드

[마이페이지] ───────────────────────────
├── 내 정보
│   ├── 기본 정보 조회
│   ├── 연락처 수정
│   └── 비상연락처
├── 보안 설정
│   ├── 비밀번호 변경
│   └── 생체인증 설정
├── 알림 설정
└── 로그아웃
```

## 2. 관리자용 웹 (Admin Dashboard)

```
[대시보드] ─────────────────────────────
├── 오늘의 현황
│   ├── 출근 현황 (출근/미출근/휴가)
│   ├── 결재 대기 건수
│   └── 이번 달 신규 입사자
├── 주요 알림
└── 통계 차트
    ├── 부서별 근무시간
    └── 월별 휴가 사용률

[직원관리] ─────────────────────────────
├── 직원 목록
│   ├── 검색/필터
│   ├── 부서별 조회
│   └── 재직/퇴사 필터
├── 직원 상세
│   ├── 기본 정보
│   ├── 근로 계약 정보
│   ├── 급여 정보
│   └── 근태 이력
├── 신규 직원 등록
│   └── 초대 링크 발송
└── 조직도 관리
    ├── 부서 관리
    └── 직급 관리

[근태관리] ─────────────────────────────
├── 일일 근태 현황
│   ├── 실시간 출퇴근 모니터링
│   └── 이상 근태 알림
├── 근태 기록 조회
│   ├── 기간별 조회
│   ├── 부서별 조회
│   └── Excel 다운로드
├── 근태 수정 요청 처리
├── 52시간 관리
│   ├── 주 52시간 초과 위험 직원
│   └── 부서별 평균 근무시간
└── 근무지 설정
    ├── GPS 좌표 등록
    └── Wi-Fi SSID 등록

[휴가관리] ─────────────────────────────
├── 휴가 결재
│   ├── 대기 목록
│   └── 처리 이력
├── 연차 현황
│   ├── 직원별 잔여 연차
│   └── 연차 촉진 대상자
└── 휴가 정책 설정
    ├── 연차 자동 부여 규칙
    └── 특별 휴가 설정

[전자계약] ─────────────────────────────
├── 계약서 템플릿 관리
│   ├── 근로계약서 템플릿
│   ├── 비밀유지서약서 템플릿
│   └── 커스텀 템플릿 생성
├── 계약 발송
│   ├── 개별 발송
│   └── 일괄 발송
├── 계약 현황
│   ├── 대기 중
│   ├── 완료
│   └── 만료 예정
└── 계약서 보관함

[급여관리] ─────────────────────────────
├── 급여 산정
│   ├── 월별 급여 계산
│   ├── 근태 데이터 연동
│   └── 수당/공제 설정
├── 급여 명세서
│   ├── 명세서 생성
│   ├── 일괄 발송
│   └── 발송 이력
├── 급여 대장
│   └── Excel 다운로드
└── 설정
    ├── 급여 항목 관리
    ├── 세율 설정
    └── 4대보험 요율

[증명서관리] ───────────────────────────
├── 발급 요청
│   ├── 대기 목록
│   └── 처리 이력
├── 증명서 템플릿
└── 자동 발급 설정

[설정] ─────────────────────────────────
├── 회사 정보
├── 관리자 계정 관리
├── 권한 설정 (RBAC)
│   ├── 역할 관리
│   └── 권한 할당
├── 알림 설정
└── 시스템 로그
```

## 3. 사용자 플로우 (User Flow)

### 3.1 신규 입사자 온보딩 플로우
```
[관리자]                          [신규 입사자]
   │                                   │
   ├─ 직원 정보 등록                    │
   ├─ 계약서 발송 ──────────────────────┤
   │                                   ├─ 앱 설치 & 초대 링크 클릭
   │                                   ├─ 본인 인증
   │                                   ├─ 계약서 확인
   │                                   ├─ 전자서명
   │                                   └─ 제출 완료
   │                                   │
   ├─ 서명 완료 알림 수신 ◀─────────────┤
   ├─ 계약서 확인 & 보관                │
   └─ 온보딩 완료                       └─ 정상 출근 가능
```

### 3.2 일일 출퇴근 플로우
```
[직원 앱]
   │
   ├─ 앱 실행
   ├─ 생체인증/PIN 로그인
   ├─ 홈 화면 > 출근 버튼
   │      │
   │      ├─ GPS 위치 확인
   │      ├─ 허용 범위 내? ─── No ──▶ "근무지 범위 밖입니다" 알림
   │      │      │
   │      │     Yes
   │      │      │
   │      └─ 출근 기록 저장 ──▶ 출근 완료 화면
   │
   └─ (퇴근 시 동일 플로우)
```

### 3.3 휴가 신청 플로우
```
[직원]                    [팀장/관리자]
   │                           │
   ├─ 휴가 신청서 작성          │
   │   ├─ 날짜 선택             │
   │   ├─ 휴가 종류 선택        │
   │   └─ 사유 입력             │
   │                           │
   ├─ 제출 ────────────────────▶ 결재 요청 알림
   │                           ├─ 신청 내용 확인
   │                           ├─ 승인/반려
   │                           │
   ◀─ 결과 알림 ────────────────┤
   │                           │
   └─ 승인 시 연차 차감         └─ 근태 현황 자동 반영
```
